# AI_Object_Detection
A real-time computer vision project demonstrating object detection and tracking capabilities using the YOLOv8 (You Only Look Once) architecture. This implementation showcases how modern deep learning models can identify and track objects in images, video files, and live webcam feeds with high accuracy and speed.

Overview:

This project leverages Ultralytics YOLOv8, a state-of-the-art object detection model, to perform:
1.Static Image Analysis: Detect and classify objects in single-frame images.
2.Video Tracking: Follow objects across multiple frames with unique identifiers.
3.Real-time Webcam Detection: Live object detection and tracking from camera feeds.

The system uses the lightweight YOLOv8n (Nano) variant, optimized for speed while maintaining reliable detection performance across 80 pre-trained object categories.
